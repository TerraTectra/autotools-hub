# CRM Pipeline Audit

Локальный анализатор CSV-выгрузки CRM. Не требует API-ключей и доступа к аккаунту клиента.

## Что выявляет

- сделки без изменений дольше заданного срока;
- сделки без следующей задачи;
- просроченные задачи;
- сумму активных сделок в зоне риска;
- нагрузку и проблемные показатели по менеджерам;
- список сделок, требующих немедленного внимания.

Результат: автономный HTML-отчёт и JSON с расчётами.

## Поддерживаемый формат

Инструмент распознаёт распространённые названия колонок: `id`, `lead_id`, `deal_id`, `name`, `title`, `status`, `stage_name`, `owner`, `responsible_name`, `amount`, `price`, `created_at`, `updated_at`, `next_task_at`, `closest_task_at`.

Перед анализом персональные данные можно удалить. Для отчёта достаточно ID, этапа, ответственного, суммы и дат.

## Запуск

```bash
node src/index.js --input sample/leads.csv --output output/audit-report.html
```

Настройка порогов:

```bash
node src/index.js --input leads.csv --stale-days 7 --no-task-days 2
```

Проверка:

```bash
npm run check
```

## Коммерческий формат

Экспресс-аудит CRM-воронки:

- обработка обезличенной CSV-выгрузки;
- HTML-отчёт;
- таблица проблемных сделок;
- выводы по менеджерам и этапам;
- список из 5–10 конкретных рекомендаций.

Ориентир: **5 000 ₽, один рабочий день**. Расширенный аудит с адаптацией колонок, сравнением периодов и индивидуальными правилами — от 10 000 ₽.
