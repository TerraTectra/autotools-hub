# CRM Data Cleaner

Локальный инструмент для очистки и объединения CRM, маркетинговых и лидогенерационных CSV-баз.

## Что делает

- объединяет несколько CSV-файлов;
- приводит email к нижнему регистру и проверяет формат;
- нормализует российские телефонные номера в формат `+7...`;
- находит дубли по email или телефону;
- объединяет непустые поля дублирующихся записей;
- сохраняет список исходных файлов и число объединённых дублей;
- формирует JSON-отчёт о качестве данных.

Инструмент работает локально и не отправляет данные во внешние сервисы.

## Запуск

```bash
node src/index.js \
  --input sample/leads-a.csv \
  --input sample/leads-b.csv \
  --output output/cleaned.csv \
  --report output/report.json
```

Проверка:

```bash
npm run check
```

## Коммерческий формат

### Очистка базы до 10 000 строк

**3 000 ₽, один рабочий день.**

Включает нормализацию телефонов/email, удаление дублей, объединение полей и отчёт качества.

### База от 10 000 до 100 000 строк

**7 000 ₽.**

Цена для более крупных объёмов и нестандартных правил рассчитывается отдельно.

### White-label для агентств

**2 500 ₽ за одну базу до 10 000 строк.** Агентство получает результат без публичного упоминания исполнителя.

### Внутренняя лицензия

**12 000 ₽.** Включает исходный код, инструкцию и письменную адаптацию одного формата колонок.

Для обработки персональных данных заказчик должен использовать законное основание и безопасный способ передачи. Публичный репозиторий не содержит клиентских данных.
