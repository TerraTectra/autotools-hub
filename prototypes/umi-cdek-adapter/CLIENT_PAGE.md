# Материал для клиента: UMI.CMS → СДЭК

Страница для пересылки владельцу интернет-магазина:

**https://terratectra.github.io/autotools-hub/site/umi-cdek-client.html**

Она описывает:

- базовый расчёт доставки и выбор пункта выдачи;
- расширенный обмен отправлениями;
- ориентировочные сроки;
- порядок подключения на тестовой копии;
- список вопросов для предварительной оценки.

На странице нет подрядных цен, прямой формы заказа или контактов разработчика. Финальную стоимость, наценку и договор с конечным клиентом определяет веб-студия.

## Проверка перед пересылкой

```bash
python scripts/check_umi_cdek_client_page.py
```

Проверка подтверждает наличие описания и опросника, а также останавливается, если на странице появляются прямые контакты разработчика, подрядные цены или внешние ссылки.
