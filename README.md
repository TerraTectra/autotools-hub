# TerraTectra Automation Solutions

Готовые прототипы интеграций и операционной автоматизации для CRM, маркетинга, интернет-магазинов и отделов продаж.

Репозиторий содержит запускаемые proof of concept: исходный код, конфигурацию окружения, Dockerfile, тесты и GitHub Actions. Каждый прототип можно адаптировать под конкретный аккаунт и довести до коммерческого пилота.

## CRM Quick Check

Браузерная бесплатная проверка CSV без загрузки файла на сервер.

- анализ выполняется локально в браузере;
- автоматическое определение разделителя CSV;
- дубли email и телефонов;
- невалидные контакты и строки без способа связи;
- зависшие сделки и отсутствие следующей задачи;
- просроченные задачи и ориентировочная сумма под риском;
- демо-режим, адаптивный интерфейс и автоматические тесты.

Статическая страница подготовлена к публикации в GitHub Pages. После однократного включения источника GitHub Actions используется workflow `Deploy public site to GitHub Pages`.

Партнёрский модуль уведомлений в Telegram при изменении статуса заказа интернет-магазина на UMI.CMS.

- штатное событие `order-status-changed`;

## Формат работы

1. Письменно фиксируются сценарий и критерии приёмки.
2. Решение адаптируется под тестовый аккаунт или обезличенную выгрузку.
3. Выполняются тесты на обезличенных данных.
4. Передаются код, инструкция и результаты проверки.
5. Production-развёртывание согласовывается отдельно.

Работа ведётся через GitHub. Секреты и персональные данные не публикуются. Контакт для обсуждения указан в профиле владельца репозитория.
