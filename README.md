# TerraTectra Automation Solutions

Готовые прототипы интеграций и операционной автоматизации для CRM, маркетинга и отделов продаж.

Репозиторий содержит запускаемые proof of concept: исходный код, конфигурацию окружения, Dockerfile, тесты и GitHub Actions. Каждый прототип можно адаптировать под конкретный аккаунт и довести до коммерческого пилота.

**Бесплатная проверка:** краткая диагностика одной обезличенной CSV-выборки до 100 строк. Полный аудит — 5 000 рублей за один рабочий день.

## CRM Pipeline Audit

Экспресс-аудит отдела продаж по обычной обезличенной CSV-выгрузке. Не требует API-ключей и доступа к CRM.

- зависшие сделки и сделки без следующей задачи;
- просроченные задачи;
- сумма активных сделок в зоне риска;
- показатели по менеджерам;
- автономный HTML-отчёт и JSON;
- поддержка типовых колонок amoCRM, Bitrix24, RetailCRM и других CRM;
- Docker, тесты и CI.

[Открыть прототип](prototypes/crm-pipeline-audit)

Экспресс-аудит: 5 000 рублей, один рабочий день. Расширенный аудит с адаптацией колонок, сравнением периодов и индивидуальными правилами — от 10 000 рублей.

## Формат работы

1. Письменно фиксируются сценарий и критерии приёмки.
2. Решение адаптируется под тестовый аккаунт или обезличенную выгрузку.
3. Выполняются тесты на обезличенных данных.
4. Передаются код, инструкция и результаты проверки.
5. Production-развёртывание согласовывается отдельно.

Работа ведётся через GitHub. Секреты и персональные данные не публикуются. Контакт для обсуждения указан в профиле владельца репозитория.
