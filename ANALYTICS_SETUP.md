# Инструкция по Настройке Аналитики

## Google Analytics 4

### Шаг 1: Создание аккаунта GA4
1. Перейдите на https://analytics.google.com/
2. Нажмите "Начать работу"
3. Введите название аккаунта: "AutoTools Hub"
4. Выберите "Создать свойство"
5. Введите название свойства: "AutoTools Hub Website"
6. Выберите часовой пояс и валюту
7. Нажмите "Далее" и выберите бизнес-цели
8. Нажмите "Создать"

### Шаг 2: Получение кода отслеживания
1. После создания свойства появится окно с кодом отслеживания
2. Скопируйте код (он начинается с `<!-- Google tag (gtag.js) -->`)
3. Код будет выглядеть примерно так:
```html
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());

  gtag('config', 'G-XXXXXXXXXX');
</script>
```

### Шаг 3: Добавление кода на сайт
1. Откройте `index.html` в редакторе
2. Вставьте код отслеживания в секцию `<head>`, после Yandex.Metrica кода
3. Повторите для всех HTML файлов (compress.html, merge.html, qr.html и т.д.)

### Шаг 4: Проверка установки
1. Установите расширение Google Analytics Debugger для Chrome
2. Откройте сайт и проверьте консоль браузера
3. Или используйте Realtime отчёт в GA4 для проверки

## Google Search Console

### Шаг 1: Добавление сайта
1. Перейдите на https://search.google.com/search-console/
2. Нажмите "Добавить ресурс"
3. Выберите префикс URL: `https://terratectra.github.io/autotools-hub/`
4. Нажмите "Продолжить"

### Шаг 2: Подтверждение владения
1. Выберите метод подтверждения "HTML-тег"
2. Скопируйте мета-тег
3. Добавьте его в секцию `<head>` всех HTML файлов
4. Нажмите "Подтвердить"

### Шаг 3: Отправка sitemap
1. В Search Console перейдите в "Sitemaps"
2. Введите URL карты сайта: `https://terratectra.github.io/autotools-hub/sitemap.xml`
3. Нажмите "Отправить"

### Шаг 4: Мониторинг
1. Проверьте "Обзор" для индексации страниц
2. Проверьте "Эффективность" для поисковых запросов
3. Проверьте "Улучшения" для проблем с SEO

## Яндекс Вебмастер

### Шаг 1: Добавление сайта
1. Перейдите на https://webmaster.yandex.ru/
2. Нажмите "Добавить сайт"
3. Введите адрес: `https://terratectra.github.io/autotools-hub/`
4. Нажмите "Добавить"

### Шаг 2: Подтверждение владения
1. Выберите метод подтверждения "Мета-тег"
2. Скопируйте мета-тег
3. Добавьте его в секцию `<head>` всех HTML файлов
4. Нажмите "Подтвердить"

### Шаг 3: Настройка индексирования
1. Перейдите в "Индексирование" → "Переобход"
2. Добавьте основные страницы для переобхода
3. Проверьте "Исключённые страницы" для проблем

### Шаг 4: Отправка sitemap
1. Перейдите в "Индексирование" → "Файлы Sitemap"
2. Добавьте URL: `https://terratectra.github.io/autotools-hub/sitemap.xml`
3. Нажмите "Добавить"

## Метрики для отслеживания

### Основные метрики
- **Пользователи:** Уникальные посетители
- **Сессии:** Количество визитов
- **Просмотры страниц:** Количество просмотров
- **Показатель отказов:** Процент пользователей, покинувших сайт после одной страницы
- **Средняя длительность сессии:** Время на сайте
- **Конверсии:** Использование инструментов

### Источники трафика
- **Organic Search:** Поисковые системы
- **Direct:** Прямой заход
- **Referral:** С других сайтов
- **Social:** Социальные сети
- **Email:** Email-рассылки

### Поведенческие метрики
- **Самые популярные инструменты:** Какие инструменты используют чаще
- **Пути пользователей:** Как пользователи перемещаются по сайту
- **Устройства:** Desktop vs Mobile
- **Браузеры:** Chrome, Firefox, Safari и т.д.
- **География:** Страны и города

## Следующие шаги после настройки

1. **Дайте время на сбор данных** (минимум 1-2 недели)
2. **Анализируйте источники трафика** - откуда приходят пользователи
3. **Оптимизируйте популярные инструменты** - улучшайте то, что используют
4. **Работайте с непопулярными инструментами** - улучшайте навигацию и описание
5. **A/B тестирование** - тестируйте изменения для улучшения конверсии
6. **Регулярный мониторинг** - проверяйте отчёты каждую неделю

## Проблемы и решения

### Низкий трафик
- **Причина:** Сайт не индексируется или низкие позиции в поиске
- **Решение:** Проверьте индексацию в Search Console, улучшите SEO, создайте контент

### Высокий показатель отказов
- **Причина:** Пользователи не находят нужное или сайт загружается медленно
- **Решение:** Улучшите навигацию, оптимизируйте скорость загрузки, добавьте призывы к действию

### Низкая конверсия
- **Причина:** Инструменты неудобны или не понятны
- **Решение:** Улучшите UX, добавьте туториалы, упростите интерфейс
