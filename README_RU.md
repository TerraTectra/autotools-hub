# AutoTools Hub - Адаптация для РФ

Набор бесплатных онлайн-инструментов, адаптированный для работы на территории Российской Федерации.

## Особенности адаптации для РФ

### ✅ Заменены сервисы на российские аналоги:
- **Google Analytics** → **Яндекс.Метрика**
- **Google AdSense** → **РСЯ** (Рекламная сеть Яндекса)
- **MetaMask/крипто** → **ЮKassa, СберPay, СБП**
- **Stripe/PayPal** → **Российские платёжные системы**

### 🔧 Технические изменения:
- Удалены заблокированные в РФ Google сервисы
- Интегрирована Яндекс.Метрика для аналитики
- Создана страница донатов с российскими платёжными системами
- Добавлена поддержка СБП (Система быстрых платежей)
- Обновлена SEO оптимизация для Яндекса

## Настройка проекта

### 1. Яндекс.Метрика

Замените `XXXXXXXX` на ваш номер счётчика Яндекс.Метрики в файлах:
- `index.html`
- Других HTML файлах (при необходимости)

Для получения счётчика:
1. Перейдите на [metrika.yandex.ru](https://metrika.yandex.ru)
2. Зарегистрируйтесь или войдите
3. Создайте новый счётчик
4. Скопируйте номер счётчика
5. Замените `XXXXXXXX` в коде

### 2. Донаты и платёжные системы

#### СБП (Система быстрых платежей)
В файле `donate.html` замените:
```html
<div class="phone-number">+7 XXX XXX XX XX</div>
```
на ваш номер телефона для приёма переводов.

#### Банковские реквизиты
В файле `donate.html` замените заглушки:
```html
<p><strong>Сбербанк:</strong> XXXX XXXX XXXX XXXX</p>
<p><strong>Тинькофф:</strong> XXXX XXXX XXXX XXXX</p>
<p><strong>ВТБ:</strong> XXXX XXXX XXXX XXXX</p>
```
на ваши реальные реквизиты карт.

#### ЮKassa
Для интеграции ЮKassa:
1. Зарегистрируйтесь на [kassa.yandex.ru](https://kassa.yandex.ru)
2. Получите API ключи (shopId, secretKey)
3. Настройте серверную часть для приёма платежей
4. Обновите файл `donate.js` для обработки платежей

### 3. РСЯ (Рекламная сеть Яндекса)

Для подключения рекламы:
1. Зарегистрируйтесь на [partner.yandex.ru](https://partner.yandex.ru)
2. Добавьте сайт в РСЯ
3. Получите код рекламного блока
4. Добавьте код в соответствующие страницы

## SEO для Яндекса

### Настройка Яндекс.Вебмастер:
1. Перейдите на [webmaster.yandex.ru](https://webmaster.yandex.ru)
2. Добавьте сайт
3. Подтвердите права (HTML-файл или мета-тег)
4. Отправьте sitemap: `https://terratectra.github.io/autotools-hub/sitemap.xml`
5. Настройте индексацию

### Микроразметка:
В файле `index.html` добавлена структурированные данные schema.org для лучшего понимания контента Яндексом.
