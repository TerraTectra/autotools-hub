# Коммерческие форматы

Все проекты выполняются письменно. Созвоны и личные встречи не требуются. Работа возможна с самозанятым; секреты и персональные данные не публикуются.

Публичный код доступен для ознакомления и локальной оценки на синтетических или обезличенных данных. Production-использование, клиентские проекты, перепродажа и white-label требуют отдельного письменного разрешения согласно [LICENSE](LICENSE).

## 1. Бесплатная пробная диагностика

**До 100 строк, один раз для одной организации.**

Можно передать небольшую обезличенную CSV-выборку. В ответ предоставляется краткая диагностическая сводка:

- сколько сделок выглядит зависшими;
- сколько сделок осталось без следующей задачи;
- ориентировочная сумма в зоне риска;
- пример структуры полного отчёта.

Пробная диагностика не включает полный HTML-отчёт, индивидуальную настройку колонок и подробные рекомендации. Её задача — показать применимость инструмента к данным заказчика до оплаты.

## 2. Экспресс-аудит CRM-воронки

**5 000 ₽, один рабочий день.**

Клиент передаёт обезличенную CSV-выгрузку. В результат входят:

- HTML-отчёт;
- список зависших сделок;
- сделки без следующей задачи и с просроченными задачами;
- сумма активных сделок в зоне риска;
- показатели по менеджерам;
- 5–10 практических рекомендаций.

API-доступ и подключение к CRM не требуются.

## 3. Расширенный аудит

**От 10 000 ₽.**

Дополнительно:

- адаптация нестандартных колонок;
- сравнение двух периодов;
- индивидуальные правила по этапам и подразделениям;
- отдельные таблицы для руководителей;
- повторная проверка после исправлений.

## 4. White-label для CRM-интеграторов

**3 500 ₽ за один отчёт.**

Интегратор передаёт обезличенную выгрузку, получает готовый отчёт без публичного упоминания исполнителя и может передать его клиенту под своим брендом. Коммуникацию с конечным клиентом интегратор оставляет у себя.

## 5. Лицензия анализатора для внутреннего использования

**15 000 ₽.**

Включает:

- исходный код CRM Pipeline Audit;
- письменное право использовать инструмент внутри одной компании;
- инструкцию запуска;
- один час письменной настройки формата колонок;
- обновление для одного дополнительного формата выгрузки.

Перепродажа исходного кода, публикация копии, SaaS и использование для обслуживания сторонних клиентов не входят в базовую внутреннюю лицензию.

## Порядок начала

1. Заказчик письменно описывает задачу или присылает обезличенную выгрузку.
2. Фиксируются результат, цена, срок и объём предоставляемых прав.
3. При необходимости заказчик оформляет договор и предоставляет закрытые доступы.
4. Работа выполняется в отдельной ветке GitHub или локально с передачей результата.
5. Оплата принимается владельцем репозитория как самозанятым.
