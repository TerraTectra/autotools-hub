Готовый продукт

Не просто демонстрация

Backend уже реализует подключение RetailCRM Marketplace, безопасное хранение секретов и работу с несколькими клиентами.

01

Индивидуальный SLA

Отдельный срок для каждого статуса, магазина или процесса обработки.

02

Без дублей

Повторное уведомление не отправляется, пока состояние заказа не изменилось.

03

Приватность по умолчанию

В Telegram передаются только номер, статус и длительность задержки.

04

Marketplace lifecycle

Активация, деактивация, заморозка и смена URL аккаунта обрабатываются автоматически.

05

Шифрование секретов

API-ключи и Telegram-токены шифруются AES-256-GCM при хранении.

06

Проверяемая сборка

Unit-тесты, процессный smoke-тест, Docker и GitHub Actions.
