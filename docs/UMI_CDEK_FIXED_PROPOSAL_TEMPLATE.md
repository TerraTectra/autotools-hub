- новые функции, не перечисленные в разделе «Входит».

## Критерии приёмки

1. `[сценарий расчёта]` возвращает ожидаемые тарифы и стоимость.
2. Выбранная доставка сохраняется в заказе UMI.CMS.
3. Ошибка внешнего API не ломает корзину или заказ.
4. Повтор операции не создаёт дубль отправления.
5. Секреты отсутствуют в клиентском HTML, журнале и репозитории.
6. `[дополнительные критерии заказчика]`.
